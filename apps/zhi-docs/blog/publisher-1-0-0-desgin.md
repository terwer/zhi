## 设置页面

```
发布设置（API地址、Token等、可以探索一下Cookie模拟登陆的方案）
平台开关

```

## 自定义发布页面（可选）

- 开启了极速发布，可使用最小化配置发布，不显示自定义页面
- 未开启极速发布，点击发布到对应平台之后显示自定义发布页面
  - 修改标题
  - 自动生成别名slug、自动提取摘要、智能标签
  - 支持拉取平台已有标签，主要与智能标签合并，不能替换，支持标签二次修改
  - 支持拉取平台分类、支持修改分类
  - 支持使用图床
